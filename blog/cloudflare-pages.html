Kết nối Cloudflare Pages

📅 26/06/2026 • 🏷️ Cloudflare, Performance

Tại sao dùng Cloudflare Pages?

Cloudflare Pages cung cấp CDN toàn cầu, giúp trang web của bạn tải nhanh hơn ở mọi nơi trên thế giới. Đặc biệt miễn phí và dễ sử dụng.

Các bước kết nối

Bước 1: Đăng nhập Cloudflare Dashboard

Bước 2: Chọn Pages → Create a project

Bước 3: Kết nối với GitHub repository

Bước 4: Configure build settings

Bước 5: Deploy

Ưu điểm so với GitHub Pages

- CDN toàn cầu (GitHub Pages chỉ có edge server)

- Hành vi caching thông minh

- Cloudflare Web Analytics miễn phí

- Hỗ trợ Serverless Functions
