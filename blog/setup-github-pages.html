GitHub Pages là gì?

GitHub Pages là dịch vụ hosting trang web tĩnh miễn phí được cung cấp trực tiếp từ GitHub. Bạn có thể dễ dàng tạo blog, portfolio, hoặc documentation site.

Bước 4: Push code lên GitHub

Ưu điểm

- Miễn phí 100%

- Tích hợp Git workflow

- Custom Domain hỗ trợ
