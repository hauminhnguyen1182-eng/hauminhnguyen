Theo dõi Analytics miễn phí

📅 26/06/2026 • 🏷️ Analytics, Privacy

Cloudflare Web Analytics

Cloudflare Web Analytics là giải pháp miễn phí, bảo mật privacy để theo dõi lượt truy cập trang web. Không cần cookie, không theo dõi cá nhân.

Cách kích hoạt

Bước 1: Vào Cloudflare Dashboard → Analytics & Logs → Web Analytics

Bước 2: Bật Web Analytics cho domain

Bước 3: Thêm script vào trang web

Code snippet

Thêm đoạn code sau vào phần <head> của trang web:

<script defer src='https://static.cloudflare.com/beacon.min.js' 
        data-cf-beacon='{"token": "YOUR_TOKEN"}'></script>

Ưu điểm

- Miễn phí không giới hạn

- Không sử dụng cookie

- Không theo dõi cá nhân (privacy-first)

- Dữ liệu real-time

- Dashboard đẹp, dễ sử dụng

So sánh với Google Analytics

Tính năng Cloudflare Google Analytics
Miễn phí ✅ Không giới hạn ✅ Có giới hạn
Privacy ✅ Không cookie ❌ Cookie required
Real-time ✅ Có ✅ Có
